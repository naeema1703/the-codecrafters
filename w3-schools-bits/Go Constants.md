## What is a Constant in Go?

constant is a value that cannot be changed once declared. It is defined using the `const` keyword.

---

### Rules for Constants
- Follow the same naming rules as variables  
- Often written in UPPERCASE (by convention, not required)  
- Can be declared inside or outside functions  

### Types of Constants

#### Typed Constant

const A int = 1

#### unTyped constant

const b = 1
